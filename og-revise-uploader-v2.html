Drop or tap to upload paper JSON
From the Claude SBA/EMQ Generator project
💡 How to use:
1. Open Claude → SBA/EMQ Generator project → upload a guideline PDF/DOCX
2. It returns a paper JSON — save it as a .json file
3. Pick a category (and section), type the topic, drop the JSON, tap Upload
4. It lands in OG-Revise-Papers / Category / Section as papers__<Topic>.json and appears under that category in the PWA's 🧪 Papers section
Category & section auto-fill if present in the JSON. Re-uploading the same topic merges questions, de-duplicates, and moves it if you change its category.

From the Claude Essay Marker artifact

1. Open Claude → Essay Marker project → upload photos of your written answer
